## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| max | 属性：进度条最大值 | `number` | 100
| value | 属性：进度条当前值 | `number` | 0
| size | 属性：进度条大小 | `tiny`、`small`、`medium`、`large` | `medium`
| color | 属性：颜色 | `primary`、 `secondry`、 `success`、 `warning`、 `alert` | `primary` |
| radius | 属性：圆半径(圆形进度条内置属性) | `number` | 60
