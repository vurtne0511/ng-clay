## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| pageSize | 属性：每页显示的数据量 | `number` | `10` |
| previousLabel | 属性：上一页文字 | `string` | `Previous` |
| nextLabel | 属性：下一页文字 | `boolean` | `Next` |
| total | 属性：数据总量 | `number` | `0` |
| pageIndex | 属性：当前页索引 | `number` | `1` |
| pageChange | 事件：翻页事件 | `function` | - |
